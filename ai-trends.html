AIツールトレンド分析

AI画像生成・動画生成ツールの人気推移を自動追跡

📊 プロジェクト概要

Google TrendsとWikipedia Pageviewsを活用し、AI画像生成・動画生成ツール（計20ツール）の人気推移を自動追跡。 毎日午前3時に自動データ収集・グラフ生成を実行し、最新トレンドをリアルタイム可視化。 SQLiteデータベースで90日間のトレンドデータを管理。

Python Google Trends API Wikipedia API SQLite Matplotlib Task Scheduler GitHub Pages

🎨 AI画像生成ツール

主要なAI画像生成ツール（Midjourney, DALL-E, Stable Diffusion等）の検索トレンドとWikipediaページビュー数の推移を可視化。

Google Trends 推移（90日間）
Wikipedia Pageviews 推移（90日間）

🎬 AI動画生成ツール

主要なAI動画生成ツール（Runway, Pika, Sora, Luma AI等10ツール）の検索トレンドとWikipediaページビュー数の推移を可視化。

Wikipedia Pageviews 推移（90日間）

🔄 自動更新について

このページのグラフは毎日午前3時（JST）に自動更新されます。 最新のトレンドデータを反映するため、ページを再読み込みしてください。
