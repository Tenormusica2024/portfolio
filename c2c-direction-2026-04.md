# C2C Portfolio Direction

Updated: 2026-04-30

## 役割

Urayaha Days は、B2B受託の信用獲得サイトではなく、個人の作家性・趣味・制作姿勢を知ってもらうためのC2Cポートフォリオ。

仕事の相談も受けたいが、主な信頼形成は以下に寄せる。

- 社会的信頼性より、人間性・作風・興味の近さ
- 「この人と友達になりたい」
- 「同じ趣味や志向で話せそう」
- 「この人なら面白いものを一緒に作れそう」
- 国内外からのコラボ・軽い相談・作品起点のオファー

## B2Bとの差分

| 項目 | B2B / Ezlize | C2C / Urayaha Days |
|---|---|---|
| 主目的 | 受託開発・問い合わせ獲得 | 人間性・作品・趣味軸の接点作り |
| 信頼軸 | 実務品質、費用感、運用安全性 | 作風、思想、継続制作、趣味/志の近さ |
| 文章 | 業務・課題・費用・導入 | 作品・背景・好きなもの・制作ログ |
| 導線 | 相談、見積り、サービス詳細 | 作品を見る、話しかける、SNS/GitHub/Zenn |
| 海外対応 | 基本は国内向け | 英語併記・非同期テキスト相談の余地あり |

## 当面の更新方針

1. **最近の活動が見えるトップにする**
   - 最新作品
   - AIキャラクター/IP
   - 個人制作ログ
   - GitHub / Zenn / X への自然な導線

2. **仕事導線は軽く残す**
   - 「Hire me」より「Talk / Collaborate / Contact」寄り
   - いきなり発注ではなく、作品や趣味を見た上で声をかけられる入口にする

3. **英語は最低限併記**
   - 英語面接や英会話前提にはしない
   - 海外から作品を見た人が概要を掴める程度の英語見出し・短文を置く
   - 相談は asynchronous / text-based collaboration を許容する表現に寄せる

4. **クリエイティブB2Bは分離候補**
   - クリエイティブ軸の商用アウトプットが固まったら、Urayaha Days内で営業色を強めず、別途クリエイティブ版B2Bサイトを作る
   - その時点で、制作メニュー・料金・実績・問い合わせ導線を分離する

## 避けること

- C2CトップをB2Bのような受託営業LPにする
- 社会的信用・実務安全性を前面に出しすぎる
- 人間性や作風より、技術スタック羅列だけに寄せる
- 海外オファー狙いで、実際の英語会話力以上の期待を持たせる
- B2B / C2C / 将来のクリエイティブB2Bを1つの人格に無理に統合する

- 主要作品のOG画像と個別ページを整える
